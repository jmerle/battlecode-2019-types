Class BCAbstractRobot

The abstract class that all implementations should extend from.

All the variables and methods starting with an underscore are used to internally to process a turn and create actions.

last_offer

last_offer: number[][] | null
  • Defined in battlecode.ts:213

A 2 by 2 grid containing the last trade offers by both teams.

last_offer[Specs.RED] is the last offer made by RED and contains an array of two integers. Similarly, last_offer[Specs.BLUE] is the last offer made by BLUE.

The first value in the array of integers is the amount of karbonite and the second one is the amount of fuel. For both offers, a positive amount signifies that the resource goes from RED to BLUE.

Available for castles (always null for other units).

buildUnit

  • buildUnit(unit: number, dx: number, dy: number): BuildAction
    • Defined in battlecode.ts:327

    Build a unit of the type unit (integer, see r.unit) in the tile that is dx steps in the x direction and dy steps in the y direction from this.me. Can only build in adjacent, empty and passable tiles.

    Uses UnitSpecs.CONSTRUCTION_FUEL fuel and UnitSpecs.CONSTRUCTION_KARBONITE karbonite (depending on the constructed unit).

    Available for pilgrims, castles and churches.

    Pilgrims can only build churches. Castles and churches can only build pilgrims, crusaders, prophets and preachers.

    • dy: number

      The amount of steps away in the y direction to build

proposeTrade

  • proposeTrade(karbonite: number, fuel: number): TradeAction
    • Defined in battlecode.ts:311

    Propose a trade with the other team. karbonite and fuel need to be integers.

    For example, for RED to make the offer "I give you 10 Karbonite if you give me 10 Fuel", the parameters would be karbonite = 10 and fuel = -10 (for BLUE, the signs are reversed).

    If the proposed trade is the same as the other team's last_offer, a trade is performed, after which the last_offer of both teams will be nullified.

    Only available for castles.

    Parameters

    • karbonite: number

      The amount of karbonite to propose

    • fuel: number

      The amount of fuel to propose

    Returns TradeAction
